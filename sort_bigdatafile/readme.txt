sort_bigdatafile
================
  sort_bigdatafile - это программа которую необходимо разработать на pure C++ 17 без привлечения дополнительных внешних библиотек, компонентов или системного API

  Производительность и объем потребляемой оперативной памяти критичны
  Ограничений на используемую дисковую память нет, но оптимизация ее объема приветствуется

  sort_bigdatafile должна принимать два обязательных позиционных входных параметра
  ================================================================================
    1 | полный путь до исходного файла
    2 | полный путь до файла результата

  sort_bigdatafile должна сортировать большой объем данных, записанных в виде файла .txt
  ======================================================================================
    gen_data.txt
    ============
      |----------------------------------
      |8691380537025648159:iKKkMmVdpeTtu4
      |13704607469474053308:axWRNXbe4ZC
      |17644630804007363494:CT9XuDqIz4e
      |...
      |<key>:<value>
       =============
        key | целочисленное без знаковое число в формате unsigned long long
        ===================================================================
          ключи в файле могут повторяться
        
        value | произвольная строка ("string"), содержащая цифро-буквенные символы

    
  sort_bigdatafile должна генерировать файл sort_data.txt в котором строки отсортированы по ключу в порядке возрастания устойчивым образом
